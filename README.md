# Customer_Churn_Prediction
Creates a model that predicts churn for Telecommunications customers and tests the model.<br>
Relies on the IBM Telco Customer Churn dataset available at https://www.kaggle.com/datasets/blastchar/telco-customer-churn.
## Results ##
Accuracy: 0.79
ROC AUC 0.83
Features positively correlating with churn were the number of total charges, the number of monthly charges, and using fiber optic internet service. Features negatively correlating with churn were number of months of tenure, having two-year contracts, and using phone service.
